## ğŸ—ï¸ Component Responsibilities

### **Services Layer**

| Service | Responsibility |
|---------|---------------|
| **ApiService** | HTTP client, CRUD operations, request/response handling, error handling |
| **AuthService** | Login, register, token management, auto-refresh, session management |
| **CacheService** | TTL-based caching, SharedPreferences + Hive integration |
| **DatabaseService** | Hive database operations, box management, type adapters |
| **SyncService** | Offline queue management, background sync, connectivity monitoring |

### **Repository Layer**

| Component | Responsibility |
|-----------|---------------|
| **BaseRepository** | CRUD template, caching logic, offline queue integration, JSON conversion |

### **Controller Layer**

| Component | Responsibility |
|-----------|---------------|
| **BaseController** | Single item state management (loading, error, data) |
| **BaseListController** | List state management, pagination, refresh |
